Cloud-Native Quiz Application Deployment

AWS CI/CD · Docker · App Runner · CloudFront · S3

Designed and deployed a cloud-native quiz app with a fully automated CI/CD pipeline and containerized architecture. Users launch quizzes, answer timed questions fetched from an API, and earn scores displayed on a leaderboard — all served securely through a single custom domain via CloudFront.

🏗️ Backend containerized with Docker, built via AWS CodeBuild, pushed to ECR
🏃 Backend deployed as stateless service on AWS App Runner (auto-pulls ECR)
🪣 Frontend built with CodeBuild, artifacts hosted on Amazon S3
🌐 CloudFront CDN routes frontend (S3) and API requests (/api/*) to App Runner
🔒 Custom domain with HTTPS configured via CloudFront + DNS policies
